# Prérequis:
  - Une version 16 de Node.js

# Pour accéder au projet:

### 1- Vous devez cloner le projet sur un dossier vide.

### 2- Allez dans le dossier et lancer ```npm run build``` puis ``` npm run watch```

### 3- Accèdez au lien http://localhost:8888/
